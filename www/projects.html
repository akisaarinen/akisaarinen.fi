Here are some of my recent and not-so-recent projects. I also share code in github, and write stuff in my blog, check them out for more.

  • Project Ghettoblaster: How we built a kick-ass portable boombox. DIY electronics and carpentering by two software engineers. What could go wrong?
  • CleanReader, a browser extension for Google Chrome to make Google Reader more usable. Sources code available at GitHub. A related blog post.
  • SmartDiet, a research toolkit for studying the energy efficiency of Android applications, and the resulting publication. Part of my research at Aalto University School of Science.
  • Concurrency Control in Group Editors: Case Study in a Product Backlog Tool (PDF). My master's thesis at about concurrency control in Group Editors, in the context of a Product Backlog management tool. I found the topic very interesting. Approved with the grade 5 (excellent).
  • CCF, a library for implementing real-time collaborative software in Scala. Still used in the software for which it was originally written for, but not developed any more. Based and developed mostly as part of my Master's Thesis. The first real project I used Scala for.
  • Centripetality, Winner of Assembly Summer 2007 Game Development Competition. See video on YouTube.
  • Lullizio, a simple IRC bot written in Ruby. Shouts all kinds of useful and non-useful stuff to IRC channels, like Youtube video titles and Twitter status contents. In active personal use.
  • NetOgler, a distributed data gathering platform. Developed with C++ using the Qt framework as a summer project with Teijo Laine in 2008. Nice experiment combining Qt, networks and graph visualization, but not developed any further.
  • PoolFXP, a Java pool-game developed during a course in Object Oriented programming at HUT back in 2005. Uses IRC as back-end for multiplayer support, which is kind of cool. Haven't touched this since 2005. Please don't judge my present coding skills based on this :-)
